# PUT API DOCS
**1.ADIMN: Update User Profile**
* **URL**
    `/apiadmin/user/:id`

* **METHOD**
    `PUT`

* **PASSED OBJETCS** <br />

    required atleast one field to update

    `firstName,` <br />
    `lastName,` <br />
    ` email,` <br />
    `isAdmin,`

* **Success Response**<br />
    * **Code:** `201` <br />
      **Message:** `Profile updated` <br />
      **data:** `{ id: "user_id", firstName, lastName, email, isAdmin }`

* **Error Response**
    * **Code:** `400` <br />
      **Message:** `User Not Found` <br />

* **Required Filed**
     * **Auth Need : True**
    * **Admin Auth Need : True**

**2. User: Follow other user**
* **URL**
    `/followUser/:user_id`

* **METHOD**
    `PUT`

* **PASSED OBJETCS** <br />
    *No need to pass anything just replace :user_id with user_id which you want to update*

* **Success Response**<br />
    * **Code:** `201` <br />
      **Message:** `Following` <br />
      **data:** `{data}`

* **Error Response**
    * **Code:** `400` <br />
      **Message:** `Something went wrong` <br />

* **Required Filed**
     * **Auth Need : True**
    * **Admin Auth Need : false**
