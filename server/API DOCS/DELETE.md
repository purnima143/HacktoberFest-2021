# DELETE API DOCS
**1. DELETE question**
* **URL**
    `/api/deleteQuestion/:id`

* **METHOD**
    `DELETE`

* **URL params**
    `question_id[integer]`

* **Success Response**<br />
    * **Code:** `201` <br />
      **Message:** `question deleted` <br />

* **Error Response**
    * **Code:** `401` <br />
      **Message:** `Authorization Required` <br />

* **Required Filed**
    * **Auth Need : True**
    * **Admin Auth Need : False**
